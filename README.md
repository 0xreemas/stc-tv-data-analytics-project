
# stc tv Data Analytics Project

This project documents my journey in analyzing stc tv platform data, transforming raw data into strategic insights to support data-driven decision-making.

## 🛠 Tools & Technologies
- SQL: Data extraction, manipulation, and creating Database Views.
- Python: Predictive modeling (SARIMAX) and recommendation systems (Collaborative Filtering).
- Power BI: Designing interactive dashboards for data visualization.

## 📂 Project Structure
- /data: Contains project datasets.
- /scripts: Contains structured SQL queries and Python modeling scripts.
- /visuals: Visualizations and dashboard screenshots.

- Note: The uploaded dataset is a sample for demonstration due to file size limits.

## 🚀 Key Achievements
- Analyzed subscriber behavior and identified a segment of 62% of low-engagement users.
- Built a predictive model to forecast peak viewership periods (Fridays and Mondays).
- Developed a recommendation algorithm to enhance user experience and improve retention rates.
